# Instalacion de App Dynfi

Clonar el directorio de la siguiente forma:
```bash
git clone git@github.com:themendas/DynFi.git
```

# Instalacion de App Dynfi

Ejecutar el siguiente comando:
```bash
docker up -d
```

# Acceso APP
